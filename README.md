# Food Ordering Behaviour and Consumer Trends: A Structured Analysis of Choices and Habits

This project presents an interactive Tableau dashboard and supporting code to analyze food delivery data. It explores customer behavior, order trends, cuisine preferences, city‑wise demand, and delivery performance through data‑driven visualizations.

## 📊 Project Overview
The goal is to uncover insights into how consumers interact with food delivery platforms and identify patterns that influence ordering habits.

### Key Objectives
- Analyze order frequency and delivery performance.
- Identify popular cuisines and city‑wise demand.
- Visualize consumer trends using Tableau dashboards.
- Integrate Python scripts for data preprocessing and analysis.

## 🧩 Files in This Repository
| File | Description |
|------|--------------|
| `app.py` | Python script for data processing and analysis. |
| `food_ordering_behavior_dataset (1).csv` | Dataset containing food ordering behavior data. |
| `index.html` | Web page displaying the dashboard interface. |
| `main.css` | Stylesheet for the web interface. |
| `sheets.twb` | Tableau workbook for interactive visualizations. |
| `Demo video.mp4` | Short demo showcasing the dashboard and insights. |
